# An example using child an parent nodes


## Setup development
Clone repo.
```
git clone --recurse-submodules https://github.com/provablyprivate/cppexample.git
```

## Install deps
* Arch Linux: `sudo pacman -S poco`
* Ubuntu: `sudo apt-get install libpoco-dev`

## Building
Use makefile.
```
make
```
